What Are Learning Paths?
-------------------------

Learning paths are curated sequences of tutorials and exercises designed to take you from your current skill level to your learning goals. Each path includes:

- **Recommended order** of tutorials
- **Hands-on exercises** to practice skills
- **Real-world examples** from published research
- **Milestone projects** to demonstrate mastery

Beginner Path
--------------

**Goal**: Become comfortable with basic spatial transcriptomics analysis

**Duration**: ~2-3 weeks with regular practice

**Topics**:

- Data loading and quality control
- Basic visualization
- Simple spatial analysis
- Cell type annotation

How to Use Learning Paths
--------------------------

1. **Assess your level** - Choose the path that matches your current skills
2. **Follow the sequence** - Complete tutorials in the recommended order
3. **Practice actively** - Work through exercises with your own data
4. **Track progress** - Use milestone projects to gauge understanding
5. **Get help** - Join our community for support and discussions
