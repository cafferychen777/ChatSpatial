# Intermediate Learning Path: Mastering Spatial Analysis Methods

**Perfect for:** Users comfortable with basic spatial analysis who want to understand advanced methods and make informed decisions about analysis approaches

**Total Time Commitment:** 6-8 hours over 2-3 weeks  
**Recommended Pace:** Focus on understanding method differences and when to apply each approach

## Prerequisites Checklist

Before starting this path, you should have:

- [ ] **Completed the [Beginner Learning Path](beginner.md)** or equivalent experience
- [ ] **Comfortable with** conversational analysis through ChatSpatial  
- [ ] **Understanding of** basic cell types and spatial domains
- [ ] **Experience creating** basic visualizations
- [ ] **Biological curiosity** about deeper spatial patterns
- [ ] **Research questions** that go beyond basic exploration

## Learning Objectives

By completing this intermediate path, you will:

- **Master method selection** - Choose the right analysis approach for your research questions
- **Understand spatial statistics** - Use advanced statistical methods to quantify patterns
- **Compare analysis approaches** - Know when to use different methods and why
- **Integrate multiple analyses** - Combine different analysis types for comprehensive insights
- **Design analysis workflows** - Plan multi-step analyses for complex research questions
- **Interpret statistical significance** - Understand p-values, effect sizes, and spatial statistics

**Time:** 120-150 minutes  
**Difficulty:** ⭐⭐⭐☆☆

**What you'll achieve:**

- **Effect Sizes:** Understanding biological vs statistical significance

**Advanced Conversation Skills:**

## Method Selection Guide

### When to Use Spatial Statistics
- **Research Question:** "Are there significant spatial patterns?"
- **Best For:** Quantifying spatial relationships, identifying hotspots
- **Use Cases:** Pattern validation, statistical significance testing
- **Limitations:** Requires sufficient sample size, sensitive to preprocessing

### When to Use Spatial Enrichment
- **Research Question:** "What biological processes are spatially organized?"
- **Best For:** Connecting spatial patterns to biological functions
- **Use Cases:** Pathway analysis, functional interpretation
- **Limitations:** Dependent on pathway database completeness

### When to Use Communication Analysis
- **Research Question:** "How do cells interact spatially?"
- **Best For:** Understanding intercellular signaling networks
- **Use Cases:** Drug target identification, mechanism discovery
- **Limitations:** Database bias, indirect inference from expression

### When to Use Multi-Sample Analysis
- **Research Question:** "How do spatial patterns differ between conditions?"
- **Best For:** Comparative studies, treatment effects
- **Use Cases:** Disease vs healthy, treatment response
- **Limitations:** Technical variation, alignment challenges

## Common Intermediate Challenges

### "Which method should I use for my question?"
- **Start with:** "What specific biological question am I trying to answer?"
- **Consider:** Sample size, data quality, research goals
- **Strategy:** Use method comparison exercises to understand trade-offs

### "My different methods give different results"
- **Normal situation:** Different methods capture different aspects
- **Approach:** Look for consistent patterns across methods
- **Focus on:** Robust findings that appear in multiple analyses

### "How do I know if results are biologically meaningful?"
- **Validation:** Cross-reference with literature and databases
- **Experimental design:** Plan targeted validation experiments
- **Collaboration:** Discuss with domain experts and experimentalists
