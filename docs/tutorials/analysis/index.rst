Overview
--------

These tutorials cover specific analytical methods for spatial transcriptomics data.

What You'll Learn
-----------------

1. **Cell Type Annotation**

   - Marker-based approaches
   - Reference-based methods (Tangram, scANVI)
   - Machine learning methods

2. **Spatial Analysis**

   - Spatial enrichment analysis
   - Domain identification
   - Pattern recognition

3. **Cell Communication**

   - Ligand-receptor analysis (LIANA, CellPhoneDB)
   - Spatial interaction patterns
   - Network visualization

Prerequisites
-------------

- Completed :doc:`../core/index`
- Preprocessed spatial dataset
- Understanding of your research question
