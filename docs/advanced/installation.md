## Quick Install

```bash
pip install chatspatial[full]
```

## Quick Configure

```bash
claude mcp add chatspatial /path/to/venv/bin/python -- -m chatspatial server
```

For other clients (Codex, Claude Desktop), see [Configuration Guide](configuration.md).
