# Workflow Examples

Complete analysis workflows demonstrating real-world ChatSpatial usage.

## Available Workflows

### [Mouse Brain Visium Analysis](case_studies/mouse_brain_visium.md)
Complete workflow for analyzing mouse brain tissue using 10x Visium technology.

**Covers:**
- Loading and preprocessing Visium data
- Spatial domain identification
- Cell type annotation
- Visualization and biological interpretation

**Dataset:** Adult mouse brain coronal section
**Technology:** 10x Visium
**Time:** ~2 hours
**Level:** Beginner to Intermediate

## How to Use Workflows

1. Choose a workflow matching your research question
2. Download sample data or use your own dataset
3. Follow the conversation examples
4. Adapt the analysis to your needs
5. Modify parameters for your data
